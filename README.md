## 🧰 Solution Standard

Every solution in this repo follows the same quality bar — see [docs/SOLUTION_TEMPLATE.md](docs/SOLUTION_TEMPLATE.md):

- ✍️ **Commented code** — the *why*, not just the *what*
- ⏱️ **Time & space complexity** stated explicitly
- 💡 **Approach write-up** — brute force → optimized, with the key insight
- 🧪 **Edge cases** considered and noted

**Commit convention:** `add: 1295-find-numbers-with-even-digits [arrays, easy]`
